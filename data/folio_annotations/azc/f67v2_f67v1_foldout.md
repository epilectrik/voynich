## f67v1 (Right) - Decorative Spoke Diagram

### Layout
```
+------------------------------------------+
|           ┌───┐ ┌───┐ ┌───┐              |
|          ┌┴───┴─┴───┴─┴───┴┐            |
|         │ [X] boxed tokens  │            |  <- Outer "ring"
|         │   (decorative)    │            |     with boxes
|         └───────────────────┘            |
|              \  |  |  /                  |
|               \ | | /                    |
|           [Y]  \|||/   [Y]              |  <- 17 spokes
|                 |||                      |
|                /|||\                     |
|               / | | \                    |
|              /  |  |  \                  |
|                                          |
|            (no center)                   |
+------------------------------------------+
```

### Transcript Structure
| Placement | Tokens | Lines | Description |
|-----------|--------|-------|-------------|
| X | 15 | 12 | Outer ring (boxed tokens) |
| Y | 56 | 17 | Spokes |

### Encoding Anomaly

**f67v1 uses ONLY X and Y** - no standard R/C/S codes!

| Standard Code | f67v1 Equivalent |
|---------------|------------------|
| R (ring) | X |
| S (spoke) | Y |

This suggests the transcriber viewed this as an "informal" or non-standard diagram.

## Comparison

| Feature | f67v2 | f67v1 |
|---------|-------|-------|
| Structure | Fragmented/artistic | Spoke-focused |
| Outer element | 4 isolated groups | Boxed ring |
| Spokes | Blended with art | Clear (17 lines) |
| Center | Ambiguous cluster | Empty |
| Encoding | C with pairs (1a/1b) | Non-standard X/Y |
| Decorative art | Heavy | Moderate (boxes) |

---

## Implications

1. **Artist took liberties** with f67v2 - less constrained than typical AZC
2. **f67v1's X/Y encoding** suggests it was viewed as non-standard
3. **Decorative elements** (corner circles, boxes) don't carry text
4. **Both diagrams simpler** than their recto counterparts (f67r1/f67r2)
