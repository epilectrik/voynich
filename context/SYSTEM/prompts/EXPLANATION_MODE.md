# EXPLANATION MODE Template

Copy this block at the start of your request:

```
EXPLANATION MODE.

Question: [your question here]

Constraints:
- Return concise answer from existing documentation
- Cite constraint numbers (C###) and phase names
- DO NOT audit the project
- DO NOT propose expansions or improvements

If the answer requires reading multiple files, do so and synthesize.
```

## When to Use

- Understanding a specific claim or constraint
- Asking "how does X work?"
- Requesting a summary of existing findings
- Clarifying terminology or relationships

## What This Mode Prevents

- Treating questions as audit requests
- Proposing new analysis
- Suggesting documentation improvements
- Confusing explanation with evaluation
