# C382: Morphology Encodes Control Phase

**Tier:** 2 | **Status:** CLOSED | **Phase:** MSTAB

## Claim

Morphological composition (prefix + suffix) encodes control phase. The grammar separates MONITORING from INTERVENTION through systematic morphological constraints.

## The Two Phases

| Phase | Prefix+Suffix | LINK Proximity | Kernel Contact |
|-------|---------------|----------------|----------------|
| MONITORING | da + (-in/-l/-r) | Adjacent | <5% |
| INTERVENTION | ch/sh + (-edy/-ey) | Distant | 100% |

## Line Structure

Each line follows a stereotyped structure:

```
ENTRY → MONITORING → LINK → INTERVENTION → EXIT
```

## Evidence

- Kernel-light combinations (da, -in/-l/-r) cluster near LINK
- Kernel-heavy combinations (ch/sh, -edy/-ey) cluster away from LINK
- Prefix-suffix constraints enforce phase coherence
- Pattern extends identically to AZC
