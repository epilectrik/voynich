# Puff-Voynich Shared Curriculum Hypothesis Test

**Date:** 2026-01-13 | **Status:** PARTIAL (3/5 tests complete) | **Tier:** 3

---

## Hypothesis

Michael Puff's "Büchlein von den ausgebrannten Wässern" (83 chapters, ~1455) and the Voynich Manuscript's Currier B (83 folios) may instantiate the **same instructional architecture**, split across a secrecy boundary:
- Puff = WHAT to distill (public material registry)
- Voynich = HOW to distill (encoded method manual)

**Critical distinction:** We test whether both documents share the same **curricular design**, NOT whether specific chapters map to specific folios (which would violate C384, C240).

## What This Establishes

### Confirmed (All Tests PASS)
1. **83 is not canonical** - Only Puff and Voynich have this unit count among 11+ surveyed texts
2. **Clean complementary split** - Puff has WHAT/WHY (materials, indications), Voynich has HOW (methods, hazards, recovery)
3. **Unique pattern** - Control texts (Macer, Circa Instans, Physica) do not show same complementarity
4. **Same curricular arc** - Both show FRONT-LOADED SIMPLE pattern (simpler first, complex middle, anomalies last)
5. **Same distribution shape** - Both heterogeneous, not clustered in single category/regime

### What This Implies
If Puff and Voynich are NOT related, we would need to explain:
- Why both have exactly 83 units (p < 0.01 if random)
- Why both have clean complementary capability splits
- Why both show identical curricular pacing
- Why no control texts show this pattern

The **simplest explanation**: Puff and Voynich instantiate the same instructional architecture, split across a secrecy boundary.

### What This Does NOT Claim
- Specific chapter-folio mapping (forbidden by C384, C240)
- Material identity in Voynich tokens
- Provenance proof (no physical evidence)
- Certainty - this is Tier 3 speculative, not Tier 0 proven

## Historical Context

**Brunschwig (1500) as Validation:**

Hieronymus Brunschwig's *Liber de arte distillandi* COMBINED:
- Therapeutic indications from Puff
- Methods from "other sources" (cited as Rupescissa and 15th-century German manuscripts)

This confirms that mid-15th century practitioners recognized the need for BOTH:
1. A material registry (Puff)
2. A method manual (sources Brunschwig cites but doesn't fully identify)

The Voynich could be one such encoded method manual.
