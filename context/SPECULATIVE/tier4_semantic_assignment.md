## Phase 3: Exhaustive Entity Matching

**Date:** 2026-01-14 | **Status:** 2/3 PASS (PARTIAL)

### Objective

Test whether individual Puff chapters can be matched 1:1 to individual Voynich B folios using Brunschwig's procedural framework as a bridge.

### Methodology

1. **Build Puff signatures:** Map each of 85 chapters to expected Voynich folio signature via:
   - Category -> Brunschwig degree (FLOWER->1st, HERB->2nd, ROOT->3rd, DANGEROUS->4th)
   - Degree -> Expected regime + CEI range + escape range + hazard range

2. **Extract Voynich profiles:** For each of 83 B folios, extract regime, CEI, hazard, escape density

3. **Compute match matrix:** Score all 85x83 pairwise combinations

4. **Optimal assignment:** Greedy 1:1 matching maximizing total score

5. **Statistical validation:** Permutation test (1000 shuffles)

**[TIER 4] Conclusion:**

- But they don't describe the same 83/84 items (rejected)
