## Process Flow Pattern

Typical Brunschwig recipe follows this sequence:

```
1. GATHER/COLLECT/OBTAIN  →  Acquire material
2. WASH/CLEAN             →  Remove dirt/impurities
3. CHOP/POUND/CRUSH       →  Mechanical breakdown
4. [STEEP/MACERATE]       →  Optional pre-treatment
5. DISTILL                →  Heat extraction
6. [REDISTILL/REFINE]     →  Optional purification
```
